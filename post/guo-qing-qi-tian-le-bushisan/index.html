国庆七天乐‘bushi’（三）

2021-10-03 3 min read # 国庆

F - Hopscotch 50(跳房子)
题意：给你一个n*n的矩阵和一个k，这个矩阵里面的数是1~k随机的，问你有没有一个从1一直跳到k的最短Manhattan distance（曼哈顿距离）,没有输出-1，有的话输出这个距离。

样例：
10 5
5 1 3 4 2 4 2 1 2 1
4 5 3 4 1 5 3 1 1 4
4 2 4 1 5 4 5 2 4 1
5 2 1 5 5 3 5 2 3 2
5 5 2 3 2 3 1 5 5 5
3 4 2 4 2 2 4 4 2 3
1 5 1 1 2 5 4 1 5 3
2 2 4 1 2 5 1 4 3 5
5 3 2 1 4 3 5 2 3 1
3 4 2 5 2 5 3 4 4 2

5

初始想法：dfs

  • 找每一个1的位置，每一个2的位置·······，dfs挨个搜；

误区：没估计时间复杂度，

正确想法：

  • 一层循环计算本层某一坐标和上一层所有的坐标的最小距离；
  • 第二层循环计算第一步中”本层“的所有坐标的最小距离，每一个距离存入数组中；
  • 第三层循环这一层数
#include <bits/stdc++.h>
using namespace std;

int n,k,x;
int dis[3000][3000];
vector<pair<int, int> > xy[3000];

int main()
{
	cin>>n>>k;
	if(k==1)
	{
		printf("0");
		return 0;
	}
	for(int i=0;i<=2500;i++)
	{
		for(int j=0;j<=2500;j++)
		{
			dis[i][j] = 0x3f3f3f3f;
		}
	}
	for(int i=1;i<=n;i++)
	{
		for(int j=1;j<=n;j++)
		{
			scanf("%d",&x);
			xy[x].push_back({i,j});
		}
	}
	for(int i=1;i<=k;i++)
	{
		if(!xy[i].size())
		{
			printf("-1");
			return 0;
		}
	}
	for(int i=2;i<=k;i++)
	{
		for(int j=0;j<xy[i-1].size();j++)
		{
			for(int l=0;l<xy[i].size();l++)
			{
				if(i!=2)
				dis[i][l]=min(dis[i][l],dis[i-1][j]+abs(xy[i][l].first-xy[i-1][j].first)+abs(xy[i][l].second-xy[i-1][j].second));
				else 
				dis[i][l]=min(dis[i][l],abs(xy[i][l].first-xy[i-1][j].first)+abs(xy[i][l].second-xy[i-1][j].second));
			}
		}
	}
	int ans=0x3f3f3f3f;
	for(int i=0;i<xy[k].size();i++)
	{
		ans=min(ans,dis[k][i]);
	}
	cout<<ans;
	return 0;
}
